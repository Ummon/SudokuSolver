# SudokuSolver
A graphical sudoku solver written in F#.

# Clone, Compile and Run

## Ubuntu

```
> git clone git@github.com:Ummon/SudokuSolver.git
> cd SudokuSolver
> ./dependencies.sh
> ./build_run.sh
```
