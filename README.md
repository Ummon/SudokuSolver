# SudokuSolver
A graphical sudoku solver written in F#.

# Clone, Compile and Run

## Ubuntu

```
> sudo apt-get install git fsharp
> git clone https://github.com/Ummon/SudokuSolver.git
> cd SudokuSolver
> ./dependencies.sh
> ./build_run.sh
```
